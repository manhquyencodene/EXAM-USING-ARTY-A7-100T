# Bài tập sử dụng Arty A7 100T

Repo này chứa các bài tập thực hành với board **Arty A7 100T** (Xilinx Artix-7 FPGA). Các mã nguồn được viết bằng Verilog/VHDL, sử dụng công cụ Vivado.

## Mục tiêu
- Làm quen với quy trình thiết kế số trên FPGA.
- Thực hành các mô hình thiết kế cơ bản như: đèn LED, nút nhấn, thanh ghi dịch, bộ đếm, giao tiếp UART, SPI,...
- Xây dựng các module tái sử dụng và testbench.

## Board sử dụng
- **Arty A7 100T** (xc7a100tcsg324-1)
- Các ngoại vi onboard: LED, nút nhấn, switch, USB-UART, DDR3, Ethernet, PMOD, ...

## Yêu cầu phần mềm
- Xilinx Vivado (2020.1 hoặc mới hơn, bản WebPack đủ dùng)
- Git (để clone repo)
